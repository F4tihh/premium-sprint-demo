# Premium Sprint Demo UI

1080 Sprint benzeri premium demo web arayüzü.

## Kurulum

```bash
npm install
npm run dev
```

Tarayıcıda genelde şu adreste açılır:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Vercel Deploy

GitHub'a yükleyip Vercel'de import ederek yayınlayabilirsiniz.

## En çok değiştireceğiniz dosya

```text
src/App.jsx
```

Renkler ve genel CSS:

```text
src/style.css
```
